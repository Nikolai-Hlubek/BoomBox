# BoomBox
Arduino mp3 player for children


## Prepare SD-Card

The <patches.053> file goes into the root folder. 

Music goes to folders called 
Music1
Music2
Music3


## Button layout

1  2  3
4  5  6

1  play
2  pause
3  stop
4  cycle through folder Music1
5  cycle through folder Music2
6  cycle through folder Music3
